Editor Keybinds

The other general keybinds can be found in the menu bar next to their buttons.

Mod = Ctrl or Cmd based on your platform

Action Keybind
Undo Mod-Z
Redo Shift-Mod-Z or Mod-Y
Bold Mod-B
Italic Mod-i
Toggle inline code Mod-/
Change line to block quote Mod->
Add row after cursor in table Mod-Enter / Shift-Enter
Escape code block Mod-Enter / Shift-Enter
Make Horizontal Rule Mod-_
Undo Mod-Z
Redo Mod-Y
